# akan-name-generator

Akan Name Generator

    Description

This is a simple web application that generates an Akan name based on a user's date of birth and gender. Akan names are derived from the day of the week a person is born and are commonly used in Ghana.

---

    Objectives

- To collect user input (date of birth and gender)
- To calculate the day of the week from the given date
- To assign an Akan name based on gender and day
- To display the result dynamically on the webpage

---

    Technologies Used

- HTML (for structure)
- CSS (for styling)
- JavaScript (for logic and functionality)

---

    How It Works

1. User enters their date of birth
2. User selects their gender
3. User clicks the "Get Akan Name" button
4. The system calculates the day of the week
5. The corresponding Akan name is displayed

---

    Project Structure

- index.html → Main webpage
- style.css → Styling file
- script.js → JavaScript logic

---

    Validation

- Ensures that the user enters a date
- Ensures that the user selects a gender
- Displays an error message if inputs are missing

---

    Features

- Simple and user-friendly interface
- Instant results without page reload
- Responsive design
